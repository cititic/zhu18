### FAQ
- Q：支持哪些时间
- A：支持年/月/周/日/时/分这些时间

- Q：如何在每分钟弹出提醒
- A：时间输入每分钟

- Q：如何在每小时开始第一分钟提醒休息眼睛
- A：时间输入1分钟

- Q：如何在每天工作时间每两小时开始第一分钟提醒休息眼睛
- A：假设每天工作时间是8小时，上午从8点-12点,下午是14点到18点时间输入10,12,16,18时1分钟

- Q：同上，但是要求只在周1到周5
- A：类似的，时间输入周一到周五10,12,16,18时1分钟
